# MathViz-Pro

Система для математического моделирования и анализа
   
   Архитектурные особенности:
   
    Многопоточность/многопроцессность для вычислений

    GPU-ускорение через CuPy (опционально)

    JIT-компиляция с Numba

    Веб-сокеты для real-time визуализации

    Собственный DSL для математических выражений
